``` eof ### Principais Diferenciais Desta Versão: 1. **Estrutura Multimodal:** A aplicação diferencia **PROMs** (onde o paciente responde sobre sua percepção) de **Calculadores** (onde o clínico insere dados técnicos para obter uma probabilidade ou score de gravidade). 2. **Lógica do NCIQ (Implante Coclear):** O Nijmegen requer uma normalização dos dados (transformação para escala 0-100). Implementei o gatilho `media_percentual` no JSON para automatizar isso. 3. **Calculador de Malignidade Cervical:** Baseado no cruzamento de dados de anamnese e exame físico (Critérios PANDORA). Ele atribui pesos maiores para idade > 60, tabagismo e fixação a planos profundos (10 pontos cada), gerando um alerta de urgência para biópsia/PAAF. 4. **Calculador CPSS:** Crucial para o seguimento de pacientes com traqueostomia prolongada em leitos de internação, ajudando a diferenciar colonização de infecção ativa. 5. **Pediatria (SN-5):** Utiliza uma escala visual analógica (VAS) de 7 pontos, conforme validado por Ribeiro et al. 6. **VoiSS:** O banco de dados está preparado para os 30 itens originais, garantindo que o score total (0-120) seja fidedigno à validação de Moreti.
